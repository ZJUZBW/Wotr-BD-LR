### 关于粉碎防御：

- 正义之怒里高等隐形术基本上可以完全代替粉碎防御。只有前期能早出粉碎防御且不会拖慢其他专长成型节奏的职业，我才会推荐7~9级点出粉碎防御（比如远程射手7级点出粉碎防御搭配队友的打击士气，或者队伍有诗人的末日挽歌）；
- 合书天使道途13级出粉碎防御；诡计大师由于专长限制，要出粉碎防御一般是7级或9级，专长不够的要么就不出了；其他道途推荐15级点出粉碎防御即可（与可怖外表同步即可）。
- 不出粉碎防御影响也不大，对于菜刀，专长的意义在于扩充输出机制（加攻击距离和扩展武器类型“异种武器擅长-斩矛”，加重击范围-“精通重击”，加借机攻击次数和措手不及时能借机-“抓准时机、包抄、战斗反射”，加攻击次数-”双持系列、顺势斩系列和远程系列“，加先攻，法系菜刀的延时超魔等），而不是让你去用这种触发非常不稳定的措手不及。
- 这游戏里大部分AB都是装备、队友和buff给的，措手不及也有很多很多的实现形式（比如先攻检定比敌人高，突袭轮，敌人震慑，敌人目盲、敌人麻痹、敌人失敏、自身隐形且敌人没有反隐专长和buff、敌人睡眠、敌人畏缩）。

### 关于回合制和即时制的夹击判定（<u>关键</u>）：

- 回合制下，夹击敌人的判定为：敌人在两个队友的**威胁范围**内。
- 即时制下，夹击敌人的判定为：敌人被两个近战队友**同时攻击（就是仇恨线在同一个人身上）**。

**基于以上条件，回合制下“抓准时机”的效果大部分可以被包抄替代，所以优先级降低。**

补充说明：目前版本，由于包抄有BUG，导致无论是回合制还是即时制，近战端包抄可以完全代替抓准时机，所以敏系菜刀可以不出抓准时机了，基本上只有大力哥和远程射手点抓准时机有一定的优先级。链接：[【正义之怒】关于包抄、抓准时机和护命的机制](https://tieba.baidu.com/p/7862623142)

**介绍**：战场之神，无懈可击 --- 敌人虽众，一击皆斩！

**链接**：

- [【正义之怒】诡计大师&剑圣BD，第二弹](https://tieba.baidu.com/p/7351794948?pn=1)

**特点**：爽就完事了，不需要像其他菜刀一样担心队友辅助不行而导致自己命中不了，队伍有基本的辅助情况下，远见打击/空间斩一开，基本上没有打不中的敌人；借机流的绝对核心；全奥术表自理，不需要依靠棕毛变化师这种辅助，队友做一下延时神能和金鹰之魂之类的卷轴即可；完美打击和完美重击千万别乱开，后期一秒空奥能点，偶尔开个完美重击打BOSS就行。

**职业**：20剑圣 或者 19剑圣1鳞甲之拳 【纯职业剑圣5级锐锋的含金量很高，建议还是尽量纯职】

**推荐道途**：诡计大师

**必备武器**：光能斩矛，即+3或+5的说话武器，**这个是剑圣常态命中的保障**，打不死生物和构装体切换随便一个斩矛即可（你可以买个白板+3斩矛，上buff的时候可以切武器，把说话武器和这个白板斩矛都用一次高等魔化变成+5）

**强弱势期**：弱势期仅在8~12级拿到空间斩之前，其余时间全程强势，不存在什么划水到后期成型，至于说要拿到空间斩才厉害的，只能说还没玩懂。

**开卡**：必选人类，19力12敏12体16智8感10魅（纯职业） 或 18力14敏12体15智7感14魅（兼鳞拳+善良阵营），信仰自选，背景扒手

**技能加点（一共每级4.5个）**：运动（必须点满）、神秘（诡计大师需要这个）、世界（做饭的）、沟通（兼鳞拳的主角选的）/隐匿、0.5级使用魔法装置（菜刀都要学）

- 顺势斩系列：顺势斩、强力顺势斩（3级出最强），终势斩、精通终势斩

**关于剑圣的绝对先攻：**
完美反应可以让剑圣先攻检定自动20骰，不公平BOSS的先攻大概是37左右（除了极个别敌人，比如阳锤55先攻）。不考虑亵渎之礼的话，诡计大师剑圣的满级智调10或以上，敏调12或以上，启明星戒指+4先攻，领袖感召+6先攻，金鹰之魂+2先攻，扒手背景+2先攻，这里是36+的先攻。
所以如果其他条件都有，理论上不需要精通先攻+精通先攻（神话）就能实现绝对先攻了，不过这样不是很稳定。

**玩法思路**：

​	**圣使的【领域掌控】，事实上代表着，圣使的强度是跟主神挂钩的**。【领域掌控】可以将主领域的法术放到普通法术位里，可以将主神全部领域的法术加入到领域法术位。比如保护领域的领域能力很垃圾，但是会给树肤，我们可以选一个包含保护领域的主神（比如阿巴达、雪琳），但是不选保护领域，也可以在2环领域法术位获得这个树肤，然后升阶到34567级，这样就可以覆盖全队了。

​	由于【领域掌控】的特性，圣使要选的主神，必须包含领域能力强的领域+领域法术强的领域。

## 僧兵

**介绍**：无表职业里的骑乘巅峰，僧兵无论是后期强度还是流程上的强度，都比骑将厉害多了。僧兵可以说命中和伤害爆发方面完爆各类战士职业（伤害方面大概要在9级动物异变+骑者变巨以后起飞，这个变巨非常关键），先攻也占绝对优势，也有绝对先攻，强度超一流。

**特点**：更偏向骑乘顺势斩（回合制极强）、和骑乘整轮攻击，骑乘冲锋也还行，纯职业僧兵的特点是先攻高+绝对先攻，额外AB方面：后期气灌武装+5AB，配合决斗手套+武训共计+5AB，秩序长袍+2AB，还有常驻的额外2次满BAB攻击+爆发用的气力额外攻击。骑马的话，对中体型或以下的敌人还有额外1AB和1AC【环境】加成。

**玩法**：纯职业僧兵的玩法是，1到2级主要靠冲锋，3到5级主骑乘顺势斩+次冲锋，6级以后骑乘顺势斩和骑乘整轮攻击看情况使用，在有动物异变（可以靠卷轴）+骑者变巨之前，也可以不骑马。

**职业**：20僧兵

**推荐道途**：跟道途关系不太大，主角适合守序侧道途

**必备武器**：大砍刀<恐惧进击>（序章和第一章用） - 新月战斧<碎虫者>（第二章6级） - 斩矛<非尼安斩矛>（10级）
