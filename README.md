## 前言

### 关于几个适合主角的完美级菜刀职业

非合书道途：剑圣、神眷猎人（崇高+荣耀/战争--目前战争域BUG不能扩表）、圣职杀手（动物+幸运/疯狂/联盟）、歌者、原怒者、奥法暴徒；

### 关于粉碎防御

- 正义之怒里高等隐形术基本上可以完全代替粉碎防御。只有前期能早出粉碎防御且不会拖慢其他专长成型节奏的职业，我才会推荐7~9级点出粉碎防御（比如远程射手7级点出粉碎防御搭配队友的打击士气，或者队伍有诗人的末日挽歌）；
- 合书天使道途13级出粉碎防御；诡计大师由于专长限制，要出粉碎防御一般是7级或9级，专长不够的要么就不出了；其他道途推荐15级点出粉碎防御即可（与可怖外表同步即可）。后期专长多了不知道点什么，也可以点出粉碎防御，节约操作量。
- 1.4版本以后，心灵屏障可以让高等隐形术不再受到敌人的识破隐形、真知术的反制，打措手不及更加容易了。
- 不出粉碎防御影响也不大，对于菜刀，专长的意义在于扩充输出机制（加攻击距离和扩展武器类型“异种武器擅长-斩矛”，加重击范围-“精通重击”，加借机攻击次数和措手不及时能借机-“抓准时机、包抄、战斗反射”，加攻击次数-”双持系列、顺势斩系列和远程系列“，加先攻，法系菜刀的延时超魔等），而不是让你去用这种触发非常不稳定的措手不及。
- 这游戏里大部分AB都是装备、队友和buff给的，措手不及也有很多很多的实现形式（比如先攻检定比敌人高，突袭轮，敌人震慑，敌人目盲、敌人麻痹、敌人失敏、自身隐形且敌人没有反隐专长和buff、敌人睡眠、敌人畏缩）。

### 关于回合制和即时制的夹击判定（<u>关键</u>）：

- 回合制下，夹击敌人的判定为：敌人在两个队友的**威胁范围**内。
- 即时制下，夹击敌人的判定为：敌人被两个近战队友**同时攻击（就是仇恨线在同一个人身上）**。

**基于以上条件，回合制下“抓准时机”的效果大部分可以被包抄替代，所以优先级降低。**

补充说明：目前版本，由于包抄有BUG，导致无论是回合制还是即时制，近战端包抄可以完全代替抓准时机，所以敏系菜刀可以不出抓准时机了，基本上只有大力哥和远程射手点抓准时机有一定的优先级。链接：[【正义之怒】关于包抄、抓准时机和护命的机制](https://tieba.baidu.com/p/7862623142)

**介绍**：战场之神，无懈可击 --- 敌人虽众，一击皆斩！

**特点**：爽就完事了，不需要像其他菜刀一样担心队友辅助不行而导致自己命中不了，队伍有基本的辅助情况下，远见打击/空间斩一开，基本上没有打不中的敌人；借机流的绝对核心；全奥术表自理，不需要依靠棕毛变化师这种辅助，队友做一下延时神能和金鹰之魂之类的卷轴即可；完美打击和完美重击千万别乱开，后期一秒空奥能点，偶尔开个完美重击打BOSS就行。

**职业**：20剑圣 或者 19剑圣1鳞甲之拳 【纯职业剑圣5级锐锋的含金量很高，建议还是尽量纯职】

**强弱势期**：弱势期仅在8~12级拿到空间斩之前，其余时间全程强势，不存在什么划水到后期成型，至于说要拿到空间斩才厉害的，只能说还没玩懂。

### 魔能射手

**介绍**：附伤BUG的集大成者，原因是远程法术打击会将射线类别变为武器类别，所以可以在射线的每一个独立伤害上附加神圣公正破敌之类的附加伤害，成型以后不靠BUG也是比较强的，但是避免不了。成型时间很晚，大概要第三章11级神话3配好装备以后初露锋芒，大成型在第四章16级神话6的时候。正义之怒里满配的魔能射手，每轮的期望伤害轻松突破3000。魔能射手成型晚的缺点，需要在前期加点的时候进行弥补，比如初始20智力+20敏捷，这样前期法术位会多一些，前期的DC也不会特别低，也可以当成小法师来玩（4级以后法术唤回，1奥能点可以换1个1环法术位，4级奥能就有7点了，一环法术管饱，而且随时可以根据敌人来换法术），而且奥法点也比正常16智力开卡多2点，奥法精准加的AB也会更高。

- 魔能射手与其他玩射线的法系是不一样的：远程法术战斗会将射线的接触攻击变为武器的普通攻击（命中率肯定会变低很多），而且也吃不到很多射线装备的加成，也不能附带偷袭骰的伤害。但是远程法术战斗可以将射线的重击范围变成武器的重击范围，附带武器附魔里的神圣公正破敌之类的伤害，所以魔能射手最佳武器是弩（基础19-20的重击范围，有精通重击以后变成17-20），而且射线变为武器攻击以后，可以吃到所用武器的神话重击+1倍率以及精通精通精通重击精通+1倍率的加成。
- 魔能射手的命中基于高等隐形术+远见打击（在敌人有克制高等隐形术的时候再用远见打击）/奥法精准+光能附魔+不使用致命瞄准，所以魔能射手对敏捷和智力的要求很高。远见打击持续12秒，可以第一轮开远见打击，第二轮开奥法精准实现超高的命中率。
- 神话能力的分裂射击可以附带武器的元素伤害、神圣公正破敌、焰爆等伤害，可以触发元素狂潮。因此弩配合队友给的小幸运效果拔群，再配合灵使的卓越魔法甚至可以把射线当成AOE法术来玩。三元素+焰爆，一旦重击，分裂射击可以触发两次元素狂潮。
- 神话道途的法术书以及诡计大师UMD3阶的法术书，其中的部分法术（同时也在魔战士法表内的，包括高等法术研习扩表的部分），可以搭配魔战士的法术战斗和法术打击来使用，而且魔战士的法术唤回也可以作用于准备施法的天使/巫妖道途法术书以及诡计大师UMD3阶的法术书。
- 精通法术唤回，奥能点消耗变为1/2法术环位（向下取整），因此3环和以下的法术位都只要1奥能点就能出。
- 通过远程法术打击射出的射线，使用的是同一个攻击检定骰子，所以一旦重击了就会让第一发武器攻击和附带的每发射线全都重击。
