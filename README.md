## 前言

### 关于适合主角的职业

​	以下职业，纯职或搭配进阶职业就有很高的综合实力，且全流程平滑。至于RP（Role player），首先纯职业就是最大的RP，其次不要觉得强的就不RP、弱的才RP，不要觉得你的RP才是RP。

**法系**：法术大师＞法师（无变体-预言学派）＞研究法师博学士，圣使，先知/探寻者（这个偏向菜刀，但是本质还是要配合天使道途来施法）；

**非骑乘菜刀（更适合即时制）**：剑圣、歌者、原怒者、突变斗士；

**骑乘菜刀（更适合回合制）**：神眷猎人（崇高+荣耀/战争--目前战争域BUG不能扩表）、圣职杀手（动物+幸运/疯狂/联盟）、屠魔游侠、僧兵；

### 关于粉碎防御

- 正义之怒里7级开始高等隐形术，后期再配合共用心灵屏障基本上可以完全代替粉碎防御【注：新版本黑水敌人新增4级游荡者职业等级，导致高隐的作用缩小了，粉碎防御的高光时刻大概就是在黑水了吧】。只有前期能早出粉碎防御且不会拖慢其他专长成型节奏的职业，我才会推荐7~9级点出粉碎防御（比如远程射手7级点出粉碎防御搭配队友的打击士气，或者队伍有诗人的末日挽歌）；组队情况下，对于非杀手/战士等多专长的职业，强行出粉碎防御的收益并不是很高，因为出了粉碎防御必定导致其他专长成型速度大幅度降低 或者会导致职业本身特性没法发挥出来。
- 合书天使道途13级出粉碎防御；诡计大师由于专长限制，要出粉碎防御一般是7级或9级，专长不够的要么就不出了；其他道途推荐15级点出粉碎防御即可（与可怖外表同步即可）。后期专长多了不知道点什么，也可以点出粉碎防御，节约操作量。
- 1.4版本以后，心灵屏障可以让高等隐形术不再受到敌人的识破隐形、真知术的反制，打措手不及更加容易了。
- 不出粉碎防御影响也不大，对于菜刀，专长的意义在于扩充输出机制（加攻击距离和扩展武器类型“异种武器擅长-斩矛”，加重击范围-“精通重击”，加借机攻击次数和措手不及时能借机-“抓准时机、包抄、战斗反射”，加攻击次数-”双持系列、顺势斩系列和远程系列“，加先攻，法系菜刀的延时超魔等），而不是让你去用这种触发非常不稳定的措手不及。
- 这游戏里大部分AB都是装备、队友和buff给的，措手不及也有很多很多的实现形式（比如先攻检定比敌人高，突袭轮，敌人震慑，敌人目盲、敌人麻痹、敌人失敏、自身隐形且敌人没有反隐专长和buff、敌人睡眠、敌人畏缩）。

### 关于回合制和即时制的夹击判定（<u>关键</u>）：

- 回合制下，夹击敌人的判定为：敌人在两个队友的**威胁范围**内。
- 即时制下，夹击敌人的判定为：敌人被两个近战队友**同时攻击（就是仇恨线在同一个人身上）**。

**基于以上条件，回合制下“抓准时机”的效果大部分可以被包抄替代，所以优先级降低。**

**补充说明**：目前版本，由于包抄有BUG，导致无论是回合制还是即时制，近战端包抄可以完全代替抓准时机，所以敏系菜刀可以不出抓准时机了，基本上只有大力哥和远程射手点抓准时机有一定的优先级（或者最佳替代办法就是大力哥自带独行战术/一人成军，然后自己出抓准时机）。链接：[【正义之怒】关于包抄、抓准时机和护命的机制](https://tieba.baidu.com/p/7862623142)

**特点**：

- 魔能射手与其他玩射线的法系是不一样的：远程法术战斗会将射线的接触攻击变为武器的普通攻击（命中率肯定会变低很多），而且也吃不到很多射线装备的加成，也不能附带偷袭骰的伤害。但是远程法术战斗可以将射线的重击范围变成武器的重击范围，附带武器附魔里的神圣公正破敌之类的伤害，所以魔能射手最佳武器是弩（基础19-20的重击范围，有精通重击以后变成17-20），而且射线变为武器攻击以后，可以吃到所用武器的神话重击+1倍率以及精通精通精通重击精通+1倍率的加成。
- 魔能射手的命中基于高等隐形术+远见打击（在敌人有克制高等隐形术的时候再用远见打击）/奥法精准+光能附魔+不使用致命瞄准，所以魔能射手对敏捷和智力的要求很高。远见打击持续12秒，可以第一轮开远见打击，第二轮开奥法精准实现超高的命中率。
- ~~神话能力的分裂射击可以附带武器的元素伤害、神圣公正破敌、焰爆等伤害，可以触发元素狂潮。因此弩配合队友给的小幸运效果拔群，再配合灵使的卓越魔法甚至可以把射线当成AOE法术来玩。三元素+焰爆，一旦重击，分裂射击可以触发两次元素狂潮。~~【1.43i版本以后不行了】
- 神话道途的法术书以及诡计大师UMD3阶的法术书，其中的部分法术（同时也在魔战士法表内的，包括高等法术研习扩表的部分），可以搭配魔战士的法术战斗和法术打击来使用，而且魔战士的法术唤回也可以作用于准备施法的天使/巫妖道途法术书以及诡计大师UMD3阶的法术书。
- 精通法术唤回，奥能点消耗变为1/2法术环位（向下取整），因此3环和以下的法术位都只要1奥能点就能出。
- 通过远程法术打击射出的射线，使用的是同一个攻击检定骰子，所以一旦重击了就会让第一发武器攻击和附带的每发射线全都重击，而且可以全吃克敌机先的加成。克敌机先+远见打击，再配合队友加成，如果还打不中那你得自我反省了。
- 目前版本，法术打击会将射线强行变成跟武器的射击范围一致，省了个及远超魔。

**==推荐道途==**：

- **灵使**：神话4选卓越魔法，神话6灵动魔法，神话8惊世神力，还有精准手套+爱乌骑乘；后期灵动闪电链+地狱烈焰射线，续航拉满，AB拉满，机动力拉满，生存能力拉满，后期基本上最全能了。记得道途对话多选善良选项，才能解锁自信之心系列法术。
- **诡计大师**：神话3选择神秘1阶，神话4选择察觉2阶、神话6选使用魔法装置2阶、神话7选使用魔法装置3阶、神话8选世界2阶；射线重击+分裂射击AOE拉满，伤害最高。后期常驻个火焰烙印还能多打一段附伤。

**强弱势期**：前期当成小法师来玩，第二章眷泽城获得+1神圣轻弩以后开始有战斗力，之后11级、神话4、13级均有质变，大成型在第四章16级神话6。

**玩法解析**：

魔能射手，主要能力都在魔上面，要是玩不来射线和法术，那还不如滚去玩物理射手。魔能射手主要是弱在前期，所以只写前期过渡的玩法，中后期一路射过去就行了。有天命所归的话，前期魔法飞弹的伤害会有不错的提升。千万记得买初学者手套。

- 序章、第一章和第二章获得神圣轻弩之前，都可以当成命中率更高还能射箭的射线流残疾法师，施法的时候务必搭配奥法精准来保证命中率。因为初始智力20点，在前期也可以用用蛛网术、闪光尘、七彩喷射、油腻术之类的打打控制。

**介绍**：圣使相比牧师的强力之处在于忠诚祝福、【领域掌控】和奥法契约，这三个能力带给圣使无与伦比的辅助能力、法术可知和灵活性。而且选择强力的领域能力+领域痴迷，让圣使在辅助施法者和辅助菜刀方面都是顶级好手。如果要给一个总体评价，圣使就是一个顶级辅助+强力施法者。

**职业**：20圣使

**推荐道途**：天使（天使天堂宝剑选择顺序：铲除奸邪、持久宝剑、迅捷流光；光环顺序：不熄明灯+穿透魔光）、灵使（超能力顺序：神话4必出卓越魔法，神话6灵动魔法）、恶魔（魔相顺序-神话4魅魔相、神话5淫梦魔相，高等魔相必选空奢魔相和遁影魔相）

**强弱势期**：圣使基本上属于全流程无弱势期的职业。

**玩法思路**：

​	**圣使的【领域掌控】，事实上代表着，圣使的强度是跟主神挂钩的**。【领域掌控】可以将主领域的法术放到普通法术位里，可以将主神全部领域的法术加入到领域法术位。比如保护领域的领域能力很垃圾，但是会给树肤，我们可以选一个包含保护领域的主神（比如阿巴达、雪琳），但是不选保护领域，也可以在2环领域法术位获得这个树肤，然后升阶到34567级，这样就可以覆盖全队了。

​	由于【领域掌控】的特性，圣使要选的主神，必须包含领域能力强的领域+领域法术强的领域。

**领域能力强的领域包括**（主要考虑运用场景多的）：疯狂、幸运、崇高、守序、善良、联盟、诡计、动物

**领域法术强的领域包括**（这里主要考虑扩表的部分）：崇高（辉煌启言、英雄气概、高等英雄气概）、荣耀（祝福武器和圣剑术）、保护（树肤、水幕拒敌）、植物（树肤、共用心里屏障）、诡计（诡影杀手等）、幸运（克敌机先、群体猫之轻灵）、联盟（伟岸雄姿，这个倒是不强，但是如果是佣兵的话可以帮法师节约法术位）

## 僧兵

**介绍**：无表职业里的骑乘巅峰，僧兵无论是后期强度还是流程上的强度，都比骑将厉害多了。僧兵可以说命中和伤害爆发方面完爆各类战士职业（伤害方面大概要在9级动物异变+骑者变巨以后起飞，这个变巨非常关键），先攻也占绝对优势，也有绝对先攻，强度超一流。

**特点**：更偏向骑乘顺势斩（回合制极强）、和骑乘整轮攻击，骑乘冲锋也还行，纯职业僧兵的特点是先攻高+绝对先攻，额外AB方面：后期气灌武装+5AB，配合决斗手套+武训共计+5AB，秩序长袍+2AB，还有常驻的额外2次满BAB攻击+爆发用的气力额外攻击。骑马的话，对中体型或以下的敌人还有额外1AB和1AC【环境】加成。

**玩法**：纯职业僧兵的玩法是，1到2级主要靠冲锋，3到5级主骑乘顺势斩+次冲锋，6级以后骑乘顺势斩和骑乘整轮攻击看情况使用，在有动物异变（可以靠卷轴）+骑者变巨之前，也可以不骑马。

**职业**：20僧兵

**推荐道途**：跟道途关系不太大，主角适合守序侧道途

**必备武器**：大砍刀<恐惧进击>（序章和第一章用） - 新月战斧<碎虫者>（第二章6级） - 斩矛<非尼安斩矛>（10级）

**开卡**：必选人类，19力14敏12体10智14感10魅，信仰自选，背景扒手

**技能加点**：运动、灵巧、察觉、隐匿/自选

**神话加点推荐**：

- 神话能力备选：常备不怠，引领打击，神话伴兽，领袖感召，双持久，无穷真气

- 神话专长备选：猛力攻击（神话），精通重击（神话），精通先攻（神话），武器专攻（神话）---这个是凑数的

- 推荐加点：（主角有道途法术书，才推荐双持久；没有神话法术书就可以不出这个）

  1引领打击、2猛力攻击（神话）、3常备不怠、4精通重击（神话）、5神话伴兽、6精通先攻（神话）

**专长加点推荐**：

- 顺势斩系列：顺势斩、强力顺势斩（3级出最强），终势斩、精通终势斩
- 借机流系列：战斗反射、包抄（5到7级必出）、抓准时机（5到11级均可）、精通重击（8到11级必出） ---额外搭配神话能力常备不怠
- 粉碎防御系列：炫目武技、粉碎防御
- 其他专长：精通先攻，武器专攻，
- 骑乘系列：奋力冲刺、不屈坐骑
- 主角特有加点（因为要搭配道途法术书）：超魔（法术增效）、超魔（法术延时）等

## 猎人

### 神眷猎人

**介绍**：神眷猎人也可以叫做小圣使 / 大圣杀，可以通过领域扩表，而且本身又是游侠+德鲁伊混合表，再混上领域的法术，法表会非常好看。而且出门自带动物，选个马就能骑上了。崇高领域神眷猎人，1级强度很高，有动物伙伴、有重压之刃、有神圣恩典，伤害和AB都是非常高的。骑乘的话必选马，然后第二章9级之前，难打的地图都准备好动物异变卷轴。

后期神眷猎人可以说是自BUFF职业里法表最好看的，重压之刃、识破命门、神能、巨灵化身、回声定位和辉煌启言都是自带，而且还能给队伍补充树肤、虔诚护盾、祝福武器、共用防护能量、共用抵抗能量等BUFF。【注：因为有不少整轮动作的BUFF，所以想要爽的话可以搭配MOD：[Bubbles Ultimate Buff Bot ](https://www.nexusmods.com/pathfinderwrathoftherighteous/mods/195)使用】

**职业**：20神眷猎人 / 16神眷猎人4博学士（拿金鹰之魂/战法转换/机会主义者）

**开卡**：人类，19力12敏12体10智16感8魅 / 20力12敏11体7智17感7魅，升级+5力 / +1感4力，背景自选/奥法学者/扒手，信仰选阿巴达，领域崇高，动物伙伴选马。
