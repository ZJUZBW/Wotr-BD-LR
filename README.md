# 正义之怒Wotr主角BD搜集

## 前言

**关于几个适合主角的完美级菜刀职业**:

非合书道途：歌者、奥法暴徒、圣职杀手、剑圣、原怒者；--- 歌者偏向辅助，成型最慢，大概8级以后各项能力和本体战力会厉害起来。

**关于粉碎防御**：正义之怒里高等隐形术基本上可以完全代替粉碎防御。只有前期能早出粉碎防御且不会拖慢其他专长成型节奏的职业，我才会推荐7~9级点出粉碎防御（比如远程射手7级点出粉碎防御搭配队友的打击士气）；合书天使道途13级出粉碎防御；诡计大师由于专长限制，要出粉碎防御一般是7级或9级，专长不够的要么就不出了；其他道途推荐15级点出粉碎防御即可（与可怖外表同步即可）。

### 斩矛剑圣

**介绍**：战场之神，无懈可击 --- 敌人虽众，一击皆斩！

**链接**：

- [【正义之怒】诡计大师&剑圣BD，第二弹](https://tieba.baidu.com/p/7351794948?pn=1)

**特点**：爽就完事了，不需要像其他菜刀一样担心队友辅助不行而导致自己命中不了，队伍有基本的辅助情况下，远见打击/空间斩一开，基本上没有打不中的敌人；借机流的绝对核心；全奥术表自理，不需要依靠棕毛变化师这种辅助，队友做一下延时神能和金鹰之魂之类的卷轴即可；完美打击和完美重击千万别乱开，后期一秒空奥能点，偶尔开个完美重击打BOSS就行。

**职业**：20剑圣 或者 19剑圣1鳞甲之拳

**推荐道途**：诡计大师

**必备武器**：光能斩矛，即+3或+5的说话武器，**这个是剑圣常态命中的保障**，打不死生物和构装体切换随便一个斩矛即可

**强弱势期**：弱势期仅在8~12级拿到空间斩之前，其余时间全程强势，不存在什么划水到后期成型，至于说要拿到空间斩才厉害的，只能说还没玩懂。

**开卡**：必选人类，19力12敏12体16智8感10魅（纯职业） 或 18力14敏12体15智7感14魅（兼鳞拳+善良阵营），信仰自选，背景扒手

**技能加点**：运动（必须点满）、神秘、世界、沟通、使用魔法装置

**专长加点推荐**：

- 顺势斩系列：顺势斩、强力顺势斩（3级出最强），终势斩、精通终势斩

- 粉碎防御系列：炫目武技、粉碎防御
- 法术专长系列：超魔（延时）
- 其他专长：精通先攻，武器专攻（送的）
- 专长溢出以后的凑数专长：高等武器专攻，武器专精，高等武器专精

**关于剑圣的绝对先攻：**
完美反应可以让剑圣先攻检定自动20骰，不公平BOSS的先攻大概是37左右。不考虑亵渎晋升的话，诡计大师剑圣的满级智调10或以上，敏调12或以上，启明星戒指+4先攻，领袖感召+6先攻，金鹰之魂+2先攻，扒手背景+2先攻，这里是36+的先攻。
所以如果其他条件都有，且非不公平难度，理论上不需要精通先攻+精通先攻（神话）就能实现绝对先攻了。

### 恶魔猎杀者 --- 神爪信仰+尖钉骑士团

**介绍**：全游戏20级不考虑道途的情况下，理论上自身拥有最高AB和最高伤害（无论是否考虑附伤流BUG）的“菜刀BD模板”。玩法源于多来源宿敌叠加（BUG），游侠羁绊的共享宿敌以后队友宿敌也享受游侠的视如仇寇加成（简称蹭队友的视如仇寇），有点像雷吉尔的地狱骑士BD的超级加强版。

**链接**：

- [【正义之怒】诡计大师&剑圣BD，第二弹](https://tieba.baidu.com/p/7351794948?pn=1) ---最后一部分

**特点**：简单无脑爽而且非常强，常态AB极高，打BOSS的爆发也极强，面对集群也不会无力；

**职业**：1原怒者（天界）5突变斗士10地狱骑士（尖钉骑士团）---这里可以塞一个僧兵（虽然多一打很强，但是要轻甲或无甲，而且还有个低级马占位置，我个人不太喜欢这个），剩下的4级有很多搭配（1僧兵/鳞拳、1受诅巫师、1奥法刑肃者、1屠魔游侠；4突变斗士、3突变斗士1受诅巫师、2圣武士等）。

**推荐武器**：任意18-20基础重击范围的双手武器都行，拿斩矛是最优解，其他的巨镰<扫荡>，弯刃大刀<厄运>也能运用自如。

**技能加点**：神秘至少2级---用于进阶地狱骑士，然后就是点满沟通，剩下的可以加使用魔法装置

**专长加点推荐**：

- 顺势斩系列：顺势斩、强力顺势斩（3级出最强），终势斩、精通终势斩

**说一下圣职杀手发力期**，1级神圣恩典和研究目标，3级开始用团队专长点出精准打击获得第一个偷袭骰，4级获得职业自带1d6偷袭骰和动物伙伴三角龙，5级点出熟练偷袭者共计3d6偷袭骰+破敌武器（2d6力场伤害和2AB、2伤害）+2级研究目标（2AB和2伤害），6级团队专长点出包抄、和+2神圣恩典（2AB和2伤害）。从3级开始成长曲线几乎完美，一路无敌到后期。

.
